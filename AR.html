📥 Importa impostazioni dal file originale

copia-incolla
Se avevi già configurato questo tool aprendolo direttamente dal disco, le sue impostazioni sono salvate nel localStorage di quel file, non accessibile da AR Nova (è una regola di sicurezza del browser). Con 30 secondi di copia-incolla le porti qui dentro.
1
Apri il file originale del tool (doppio click sul file .html dal disco). Ti si apre nel browser come l'hai sempre aperto.
2
Apri la Console del browser: F12 su Windows/Linux, ⌥⌘I su Mac, poi clicca la scheda Console.
3
Incolla questo comando nella console e premi Invio. Copierà tutte le impostazioni negli appunti:
copy(JSON.stringify(Object.fromEntries(Object.keys(localStorage).map(k=>[k,localStorage.getItem(k)])),null,2))
4
Torna qui e incolla nell'area sotto, poi clicca Importa:

🔑 Porta le tue chiavi dentro il pianeta

Perché serve: il browser isola il localStorage per dominio. Quando apri il tool direttamente dal disco (doppio click) le chiavi vivono lì. Quando lo apri da AR Nova, la pagina ha un dominio blob: diverso e non vede quelle chiavi. Questo mago le copia una volta e da quel momento AR Nova le inietta ogni volta che apri il pianeta.
1Apri il file originale
Doppio click sul file file.html dal tuo disco — la stessa cosa che facevi prima di AR Nova. Deve essere lo stesso file che vedi usare normalmente.
Non sei sicuro di quale sia? È il file che contiene le chiavi. Se hai più versioni, apri quella che usi di solito e che ha le chiavi già configurate al suo interno.
2Apri la Console del browser
Nella tab dove hai aperto il tool: premi F12 (Windows/Linux) oppure ⌥+⌘+I (Mac). Si apre un pannello: clicca la scheda Console.
3Incolla ed esegui
Copia questo comando col bottone qui sotto, incollalo nella console, premi Invio. Fa due cose: ti mostra le chiavi trovate e le copia negli appunti. 
const __all = Object.fromEntries(Object.keys(localStorage).map(k=>[k,localStorage.getItem(k)]));
console.table(__all);
copy(JSON.stringify(__all,null,2));
console.log('%c✓ '+Object.keys(__all).length+' chiavi copiate negli appunti','color:lime;font-size:14px');
⚠ Se vedi 0 chiavi copiate, vuol dire che hai aperto il file sbagliato (o il tool non aveva nulla salvato). Apri la versione giusta e riprova.
4Torna qui e incolla
